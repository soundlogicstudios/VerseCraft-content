# VerseCraft Expo Minimal (v2 Launch Slate)

Minimal Expo app shell that can **run on a phone immediately** and tap through bundled story packs.

## Run
```bash
npm install
npm run start
```

## Validate all story packs
```bash
npm run validate
```

## Included titles (bundled)
- Paint the Sky (Oregon Trail)
- Code Blue (Hospital Romance)
- The Back Rooms (Liminal Horror)
- Lore of Lorecraft (Arcane Tutorial)
- Creation of the Cosmos (Mythic)
- The Crimson Seagull and the Burning Ember Crown (Epic Pirate Fantasy)
- Relic of Sylara (Relic Fantasy)
- A Wasteland Story (Post-apocalyptic)
- Classic Literature Pack (placeholder slot -> wire to Store later)
